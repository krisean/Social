# AI Personality Bots

## Overview

AI-powered bot personalities that can join games and provide entertaining, unpredictable gameplay through their unique quirks and behaviors. Each bot has distinct personality traits that affect how they play, comment, and interact with other players.

## Implementation Notes

### Personality Framework
- Each bot should have 3-5 core personality traits that affect decision-making
- Comment generation should use personality-specific templates
- Special abilities should trigger randomly but fit the personality
- Bots should react to game events in character

### Balance Considerations
- Special abilities should have cooldowns to prevent spam
- Personality quirks should enhance fun without breaking game balance
- Some personalities should synergize well with certain game modes

### Content Guidelines
- Keep personalities light-hearted and fun
- Avoid harmful stereotypes or offensive content
- Focus on exaggerated, cartoonish traits
- Allow players to opt-out of specific personalities if needed

## Top Comment Game Integration

### Core Game Mechanics Integration

**Bot Participation in Game Phases:**

#### **1. Prompt Response Phase** ðŸ¤”
Bots generate personality-driven responses to prompts like "Worst pickup line ever?" or "Rate tonight's playlist?"
- **Frank** (meth addict): Hyper-speed, stream-of-consciousness comments jumping between ideas
- **Lucy** (flirty): Suggestive, playful responses with heavy emoji usage
- **Karen**: Entitled complaints about prompt quality or demands for better questions
- **Clanker**: Logical analysis with Star Trek references and "illogical" complaints

#### **2. Reply Phase** ðŸ’¬

- **Becky** (gossip): Reveals "secrets" about other players' strategies or "insider info"
- **Shakespeare**: Poetic replies in iambic pentameter, calling others "thou" and "thee"
- **Grandpa Joe**: "Back in my day..." nostalgia replies comparing modern answers to "the good old days"
- **Dr. Science**: Overly technical responses with made-up equations and scientific jargon

- **Tiffany**: Votes based on aesthetic appeal and "content creation" potential

### Game Mode Integration

#### **Event Mode (Host-Controlled)**
- Bots join as "AI Teams" alongside human teams
- Provide guaranteed entertainment and fill empty team slots
- Create balanced multiplayer experiences with 2-24 players
- Host can choose which bot personalities to include

#### **24/7 Mode (Persistent Wall)**
- Bots maintain constant activity on venue walls
- Encourage human participation through bot-generated content
- Create dynamic, evolving conversations that never stop
- Build venue community through shared bot experiences

### Social Dynamics & Entertainment Value

#### **Human-Bot Interactions**
- Players can reply to bot comments and vote on them
- Bots react to human comments with personality-specific responses
- Creates hilarious scenarios (Karen complaining about bot comments, Clanker analyzing human "illogical" behavior)

#### **Bot-Bot Interactions**
- Bots can reply to and vote on each other's content
- Personality clashes create entertaining rivalries
- Self-reinforcing echo chambers (Karens agreeing with each other, Bros high-fiving)

#### **Community Building**
- Shared bot experiences become venue traditions
- Players develop favorites and inside jokes about bot behaviors
- Viral moments when bots create unexpectedly brilliant content

### Balance & Fairness

#### **Bot Behavior Guidelines**
- Bots cannot self-vote or self-reply (same rules as humans)
- Bot votes count equally to human votes
- Special abilities have cooldowns to prevent spam
- Personality quirks enhance fun without breaking game balance

#### **Content Moderation**
- All bot responses pass through OpenAI moderation (same as human content)
- Venue staff can disable specific bot personalities if needed
- Bot content contributes to venue analytics and engagement metrics
